🚀 GETTING STARTED IN 5 MINUTES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Get OpenAI API Key (2 minutes)
┌────────────────────────────────────────────────────────────────┐
│ 1. Go to: https://platform.openai.com/api-keys                │
│ 2. Click "Create new secret key"                              │
│ 3. Copy the key (looks like: sk-...)                          │
│ 4. ✅ You have your API key!                                  │
└────────────────────────────────────────────────────────────────┘

STEP 2: Create .env File (1 minute)
┌────────────────────────────────────────────────────────────────┐
│ Location: e:\1 Month\project 5\.env                           │
│                                                                │
│ Content:                                                       │
│   OPENAI_API_KEY=sk-your-key-here                            │
│                                                                │
│ Replace sk-your-key-here with your actual key                │
└────────────────────────────────────────────────────────────────┘

STEP 3: Install Dependencies (1 minute)
┌────────────────────────────────────────────────────────────────┐
│ Open PowerShell in project folder and run:                    │
│                                                                │
│   pip install -r requirements.txt                             │
│                                                                │
│ Wait for installation to complete                             │
└────────────────────────────────────────────────────────────────┘

STEP 4: Run Backend (1 minute)
┌────────────────────────────────────────────────────────────────┐
│ In the same PowerShell, run:                                  │
│                                                                │
│   python backend.py                                           │
│                                                                │
│ You should see:                                               │
│   "Uvicorn running on http://0.0.0.0:8000"                   │
│                                                                │
│ ✅ Backend is now running!                                   │
└────────────────────────────────────────────────────────────────┘

STEP 5: Test Chatbot (Open in Browser)
┌────────────────────────────────────────────────────────────────┐
│ 1. Double-click index.html to open in browser                 │
│ 2. Scroll to bottom-right corner                              │
│ 3. Click the 💬 button                                        │
│ 4. Type: "What are your projects?"                            │
│ 5. Press Enter                                                │
│ 6. ✅ See the chatbot response!                               │
└────────────────────────────────────────────────────────────────┘

🐛 COMMON ISSUES & FIXES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Issue: ModuleNotFoundError: No module named 'fastapi'
Fix:   pip install -r requirements.txt

Issue: OPENAI_API_KEY not found in environment variables
Fix:   1. Check .env file exists in project folder
       2. Add: OPENAI_API_KEY=sk-your-key
       3. Restart backend

Issue: Unable to connect to the assistant
Fix:   1. Ensure backend is running: python backend.py
       2. Check it's on http://localhost:8000
       3. Open browser console (F12) for details

Issue: Chatbot responds slowly
Fix:   Normal! OpenAI takes 1-3 seconds
       Check internet connection

Issue: 401 Unauthorized error
Fix:   1. Check API key is valid
       2. Go to https://platform.openai.com/account
       3. Ensure key has available credits

For more issues: Check CHATBOT_SETUP.md
