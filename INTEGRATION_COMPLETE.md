❓ FREQUENTLY ASKED QUESTIONS:

Q: I'm seeing "ModuleNotFoundError"
A: Run: pip install -r requirements.txt

Q: "OPENAI_API_KEY not found"
A: 1. Check .env file exists
   2. Add your API key to .env
   3. Restart backend

Q: Chatbot says "Unable to connect"
A: 1. Ensure backend is running: python backend.py
   2. Check it's on http://localhost:8000
   3. Check browser console (F12) for errors

Q: Backend won't start
A: 1. Python installed? (python --version)
   2. Dependencies installed? (pip install -r requirements.txt)
   3. Port 8000 available? (Change if needed)
   4. Check error message carefully

Q: Can I change what the chatbot says?
A: Yes! Edit PORTFOLIO_CONTEXT in backend.py

Q: Will this cost me a lot?
A: No! ~$0.0005 per message. Very cheap.

Q: How do I deploy this?
A: See CHATBOT_SETUP.md for Heroku, AWS, etc.
